# Chat Malawi Landing Page

A high-converting, modern landing page for the Chat Malawi Android app. This website is designed to promote the app, allow users to download the APK, track user behavior, and rank on Google.

## 🚀 Features

- **Modern Design**: Mobile-first, responsive design with blue/green theme
- **PWA Support**: Installable as a Progressive Web App
- **Analytics Tracking**: Google Analytics integration with custom event tracking
- **SEO Optimized**: Meta tags, structured data, and Open Graph tags
- **Offline Support**: Service worker for offline functionality
- **Social Sharing**: WhatsApp, Facebook, and copy link functionality
- **Trust Indicators**: Safety badges, download counters, and social proof

## 📱 Sections

1. **Hero Section**: App branding, download button, file info
2. **Trust & Safety**: Security assurances and testing info
3. **Features**: Key app features with icons
4. **Screenshots**: Phone mockups (placeholders)
5. **How to Install**: Step-by-step installation guide
6. **Social Proof**: Usage statistics and testimonials
7. **Viral Share**: Social media sharing buttons
8. **Final CTA**: Final download call-to-action

## 🛠️ Technologies Used

- **HTML5**: Semantic markup with SEO optimization
- **CSS3**: Modern styling with animations and gradients
- **JavaScript ES6+**: Progressive enhancement and analytics
- **PWA**: Manifest.json and Service Worker
- **Google Analytics**: User behavior tracking

## 🚀 Deployment

This site is designed to work on any static hosting service:

1. **GitHub Pages**: Free and easy to set up
2. **Netlify**: Continuous deployment and form handling
3. **Vercel**: Modern hosting with analytics
4. **Firebase Hosting**: Google's hosting solution
5. **Any static hosting**: Just upload the files

### Deployment Steps

1. Upload all files to your hosting provider
2. Ensure the APK file is accessible for download
3. Update Google Analytics ID in `index.html`
4. Test all functionality on mobile devices
5. Submit to search engines

## 📱 PWA Installation

Users can install this as a PWA:
1. Visit the site on a mobile browser
2. Look for the "Add to Home Screen" prompt
3. Tap install to add to home screen

## 🔧 Customization

### Update App Information
Edit `index.html` to update:
- App name and tagline
- Version number and file size
- Download counter starting number
- Social proof statistics

### Update Styling
Edit `style.css` to customize:
- Color scheme (currently blue/green)
- Typography and spacing
- Animation effects
- Responsive breakpoints

### Update Analytics
Edit `script.js` to:
- Add your Google Analytics ID
- Modify tracking events
- Add custom analytics logic
